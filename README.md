## Descripción

El proyecto consiste de una aplicación que permite realizar compras a una
pizzería. Para hacer uso de la aplicación se requiere de una cuenta, la cual
es posible crear por medio de la ventana de registro; una vez creada la cuenta
se inicia sesión y posteriormente se podrá realizar una compra dentro de la
aplicación.

## Dependencias

Para poder compilar el ejecutar el proyecto desde Visual Studio, es necesario
instalar el NuGet de System.Data.SQLite. Para realizar esta tarea hacer:

- Click en Tools > NuGet Package Manager > Manage NuGet Packages for Solution...
- Buscar System.Data.SQLite.
- Seleccionar el Proyecto "PizzaApp".
- Hacer click en "Install".
